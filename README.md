# MiniApp

A simple MERN stack app for login and logout

## Installation

Use the package manager npm to install dependicies.

```bash
miniapp $ npm install 
```
Change directory to client/
```bash
miniapp/client $ npm install  
```
## Running
 chnage directory to miniapp/
```python
miniapp/ $ npm  run dev

```
